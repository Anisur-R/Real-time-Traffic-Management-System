# 🚦 Real-time-Traffic-Management-System

A full-stack web simulation of a smart traffic control system. This project demonstrates real-time state management, asynchronous UI updates, and industrial logic implementation.

## ✨ Features
- **Real-time Simulation:** Automatic signal switching based on pre-defined timers.
- **Visual Feedback:** High-fidelity UI with "dim-to-bright" light transitions.
- **Emergency Mode:** One-click priority override for emergency vehicles (e.g., Ambulances).
- **Manual Control:** Independent control over North-South and East-West signals.

## 🛠️ Tech Stack
- **Backend:** Python (Flask)
- **Frontend:** HTML5, CSS3, JavaScript (ES6)
- **Concurrency:** JavaScript `setInterval` for real-time synchronization.
